CONTEXTE & ARCHITECTURE
================================================================================

Stack Technique:
- Frontend: Next.js 15 + React + Tailwind CSS + PWA
- Backend: Next.js API Routes (App Router)
- Database: MongoDB
- Payments: Stripe (LIVE mode)
- Email: Resend
- AI: OpenAI (GPT-4o-mini + Whisper + Vision)
- Images: Cloudinary
- Auth: JWT + bcrypt

Architecture Monolithique (2 fichiers principaux):
- /app/app/page.js (2103 lignes - TOUT le frontend)
- /app/app/api/[[...path]]/route.js (1883 lignes - TOUTE l'API)

Fonctionnalités:
✅ Auth JWT multi-utilisateurs isolés
✅ Création EDL (T1-T4)
✅ Inspection pièce par pièce (5 étapes)
✅ Upload photos + tri IA automatique
✅ Notes vocales → texte (Whisper)
✅ Paiements Stripe (9.90€, 49€/mois, 149€/mois)
✅ Génération PDF serveur (pdf-lib)
✅ Envoi email auto (Resend)
✅ Code promo TEST100 (déblocage gratuit)
✅ PWA installable

Problèmes à résoudre:
⚠️ Code très monolithique
⚠️ Photos stockées en Base64 au lieu de Cloudinary URLs
⚠️ Cache frontend après paiement
